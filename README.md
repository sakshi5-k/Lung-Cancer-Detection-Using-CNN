Lung Cancer CT Detection & Classification (Streamlit)

Setup Instructions:
-------------------
1. Install dependencies:
   pip install -r requirements.txt

2. Place model weights (optional for now):
   - CNN: models/cnn_model.h5
   - YOLO: models/yolov8n.pt

3. Run the app:
   streamlit run app.py

4. Upload CT images to see nodule detection and benign/malignant classification.

5. A dummy test image is included in uploads/ for immediate testing.
